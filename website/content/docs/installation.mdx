# Installation

## Backend (Ruby Gem)

Add Zodra to your Gemfile:

```ruby title="Gemfile"
gem "zodra"
```

Then run:

```bash
bundle install
```

## Frontend (NPM Package)

Install the TypeScript client:

```bash
npm install @zodra/client
```

Or with your preferred package manager:

```bash
pnpm add @zodra/client
yarn add @zodra/client
```
